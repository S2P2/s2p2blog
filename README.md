# S2P2 Blog

A personal technical blog built with Astro.

## Site

- Title: `S2P2`
- Author: `Sathianphong Phongsathian`
- Description: `Healthcare, code, AI, and the messy middle.`

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Project structure

- `src/pages/` — routes and top-level pages
- `src/components/` — shared UI components
- `src/layouts/` — shared layouts
- `src/content/blog/` — blog posts in Markdown/MDX
- `src/styles/global.css` — global styling
- `public/` — static assets

## Notes

- The header and footer use GitHub only for social links.
- Header GitHub: `https://github.com/S2P2`
- Footer GitHub: `https://github.com/S2P2`
- Site metadata lives in `src/consts.ts`.
